- Compare Folders插件

- VSCode中的Markdown All in One插件
为markdown文件生成目录，安装后点开md文件，然后快捷键CTRL(CMD)+SHIFT+P，输入Markdown All in One: Create Table of Contents回车即可

::: tip

1. vs code为什么一直在重新激活终端?

- 打开 VS CodeSettings
- 搜索Python Locator
- 切换native到js
- 重新启动 vs code
